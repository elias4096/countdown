# Countdown

A simple website where users can enter a specific date, and the site displays a live countdown showing the time remaining until that date.

![alt text](/images/countdown.png)

# Motivation

I wanted to experiment with building a basic web server from scratch using only raw TCP, avoiding any web frameworks.

# Gettings Started

## Prerequisites

- Rust

## Installing

```
git clone https://github.com/elias4096/countdown.git
cd countdown

# Run development server
cargo run

# Go to: 127.0.0.1:8080
```
